# GPA-Calculator-for-a-university-student
A collection of C++ programs designed to process student marks, calculate letter grades, and compute total GPA.

## Features 
** GPA Calculator ('GPA calculations.cpp')**
* Uses a 'for' loop to collect course details for multiple subjects.
* Prompts for course codes, credit hours, and numerical scores.
* Converts scores (0-100) into letter grades and quality points (0.0-5.0).
* Automatically calculates total credit hours and final weighed GPA.

  ** Grading system**
  * Evaluates student marks and assigns corresponding grade boundaries.

  ## How to Run
  * open the project in Dev C++ (or any C++ compilerP).
  * compile and run 'GPA calculations.cpp'.
  * Follow the terminal prompts to enter your course details!
